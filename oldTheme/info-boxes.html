The Info Box is a universal wrapper element for placing text, image, video, button etc. in a separated area with a different background color. You can enhance Info Box by adding the following optional classes:

  • light / dark / colored / white-bg
  • light + gradient
  • no-border
  • rounded
  • shadow
  • tight
  • action-box
  • adapted-box
