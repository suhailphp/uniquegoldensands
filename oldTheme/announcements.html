Announcements

  • Feature Boxes
  • Info Boxes

  • Tabs & Toggles
  • Buttons & Nav. Elements
  • Pricing Tables
  • Other Elements

The Announcements module can be used for displaying news and announcements in various styles and layouts. It can contain text and preview images. The module is very flexible — it allows you to create a few types of layouts using the same structure. You can arrange announcements both in one- and multicolumn layout.

Add one of the following classes to each "section" element in multicolumn layout in order to adapt the module view for screens with specified maximum width:

  • adapted-max-767px — multicolumn layout turns into one-column layout if displayed on small-screen mobile devices (screens with maximum width of 767 pixels);
  • adapted-max-959px — multicolumn layout turns into one-column layout if displayed on small-screen mobile devices and tablets in portrait mode (screens with maximum width of 959 pixels).
