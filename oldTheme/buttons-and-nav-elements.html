The Feature Menu can contain unlimited number of items, each of which has the title and subtitle, and is represented by a Font Awesome icon. You can use either default orientation (vertical) or inline (horizontal).

For inline type Feature Menu, there are 2 optional classes — "adapted-nav-left" and "adapted-nav-right" — which allows to adapt the menu for tablet portrait size (768-959 pixels wide).
